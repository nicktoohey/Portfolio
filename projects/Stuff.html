Stuff+

Developed a college baseball Stuff+ model to predict the expected run value of a pitch from its metrics, and designed post-game Stuff reports for the Springfield Drifters (WCL) to support player development for coaches and athletes.

Stuff+ Model

Used XGBoost Regression to predict the expected run value of a pitch based on Velocity, IVB, HB, Spin Rate, Spin Axis, Release Height, Release Side, Extension, adjusted Vertical and Horizontal Approach Angle, Arm Angle, and velo/IVB/HB difference from primary FB. Optuna package was used to tune the hyperparameters of the model. Trained using college Trackman data from 2023-2025. Created Post-Game Reports to send out to pitchers and coaches to assist with player development during the summer season and track progression. Developed a blob tool to evaluate how the model grades on average with a certain velocity and release height range and also compare how the model grades with arm angle as a predictor vs no arm angle predictor.
